User 是我们的一个自定义对象：
```kotlin
data class User(var name: String)
```

界面的定义如下：

```kotlin
var user by mutableStateOf(User("宏正"))  
setContent {  
    Text(text = user.name, Modifier.clickable {  
        user = User("HongZheng")  
    })  
}
```

这样，我们在更新的时候，重新 new 了一个对象出来，进行了替换，才触发了 mutableState 的状态更新，触发 recompose。
