# link 标签引入 css 文件

可以通过这样的方式引入 css 文件： 
```html
<link href="css/style.css" rel="stylesheet">
```

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title></title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <link href="css/style.css" rel="stylesheet">
  </head>
  <body>
  
  </body>
</html>
```
